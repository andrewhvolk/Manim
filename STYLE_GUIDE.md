# Visual and Pedagogical Style Guide

## Audience

Default audience: undergraduate students in mathematics or related quantitative courses.

## Teaching style

- Show one conceptual step at a time.
- Do not overload the screen.
- Prefer paced reveals over large blocks of simultaneous content.
- Keep notation standard and explicit.
- Favor clean examples over flashy transitions.

## Visual style

- Use a calm, readable layout.
- Keep backgrounds simple.
- Use color with restraint and consistency.
- Maintain strong contrast for readability.
- Keep text large enough to read comfortably in normal video playback.

## Animation style

- Keep transitions purposeful.
- Minimize unnecessary motion.
- Use emphasis to direct attention to the active mathematical idea.
- Avoid visual clutter.

## Code style

- Keep scene code readable and modular.
- Use helper functions only when they improve clarity.
- Include short comments when they clarify pedagogy or structure.
- Avoid premature abstraction in the first version.

## Mathematical style

- Use LaTeX for formulas and symbolic notation where appropriate.
- Prefer conventional notation used in college mathematics instruction.
- Make assumptions and variable definitions visible when helpful.

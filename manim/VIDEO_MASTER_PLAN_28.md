# Math 130 Test 3 — 28-Video Master Plan (Manim + Professor Narration)

## Global Production Standards

### Visual Style Guide (mandatory for all videos)
- **Background**: Deep navy gradient (`#0B1026 -> #1A1F4B`)
- **Primary math text/objects**: Cyan `#34D1FF`
- **Secondary highlights**: Violet `#9B6BFF`
- **Answer reveal**: Green `#26D07C`
- **Warnings/common mistakes**: Orange `#FF9F43`
- **Incorrect path overlay**: Coral red `#FF5A6E`
- **Body text**: Off-white `#F5F7FF`
- **Grid/axes**: Blue-gray `#607399`

### Animation Language
- Intro titles: `Write + FadeIn`
- Formula transitions: `TransformMatchingTex`
- Graph/vector builds: `Create`, `GrowArrow`
- Key step emphasis: `SurroundingRectangle + Indicate`
- Answer reveal: `Flash + color shift to green`
- Error correction: red X overlay + rewind transform

### Narration Standards
- Voice: math professor, calm and confident
- Sequence each problem: **Goal -> Setup -> Solve -> Check**
- Always state units + degree/radian mode where relevant
- End each video with 3 quick-check prompts

# Lesson Brief Guide

Use this guide when requesting a new Manim lesson. A brief should be short, specific, and easy to turn into a single lesson scene or a small lesson sequence.

## Standard brief format

Include the following sections in this order:

### 1. Lesson title

State the lesson name as it should appear in the video.

### 2. Learning objective

Describe what the student should understand or be able to do after the lesson.

### 3. Target audience

Name the expected learners, such as undergraduate students, high school algebra students, or adult learners returning to math.

### 4. Math content

List the main mathematical ideas, formulas, or steps that must appear in the lesson.

### 5. Visual style

Describe the preferred presentation style, such as clean board-style math, graph-based visuals, color highlights, or slow paced step-by-step reveals.

### 6. Example problem

Provide one representative problem that the lesson should solve or explain on screen.

### 7. Summary points

List the short takeaways that should appear near the end of the lesson.

### 8. Desired runtime

Give a target runtime, such as 20 seconds, 45 seconds, or 90 seconds.

## Filled example brief

```markdown
# Lesson brief

## Lesson title
Solving One-Step Linear Equations

## Learning objective
Help students solve equations of the form x + b = c by isolating x.

## Target audience
Beginning algebra students.

## Math content
- Meaning of isolating the variable
- Using inverse operations
- Checking the final answer

## Visual style
- Clean centered equations
- One step revealed at a time
- Light use of color to highlight the operation being undone

## Example problem
Solve x + 5 = 12.

## Summary points
- Undo addition with subtraction
- Keep both sides balanced
- Check the solution in the original equation

## Desired runtime
30 to 45 seconds
```
